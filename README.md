# c964-model

## Setup

Requirements:
- `python ~= 3.10.12`
- `pip ~= 22.0.2`

1. Create and activate virtualenv:

```bash
python -m venv .venv
source .venv/bin/activate
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Start Jupyter notebook server:

```bash
jupyter notebook ./notebooks
```
